# waterflowsensor

## Parts list
> Arduino (Nano) 
> 
> 
> Water Flow Meter (YF-S201) - Working range 1-30L/min - Water pressure <= 1.75 MPa

## Conections
> Water Flow Sensor;
> 
> Black Wire GND to ground on Nano;
> 
> Red Wire VCC to 3V on Nano;
> 
> Yellow Wire to D3 on Nano;
